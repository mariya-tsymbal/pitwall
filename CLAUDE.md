# Pitwall — Claude Context

## Project
A full-stack F1 data platform for exploring historical race results, live session timing, and AI-powered race analysis. GitHub: `mariya-tsymbal/pitwall`.

## Developer Background
Mariya is an experienced frontend developer (React/React Native) expanding into full-stack. Explanations should assume strong TypeScript/React knowledge but should be clear when introducing backend, database, or infrastructure concepts.

## Tech Stack
- **Frontend:** React, TypeScript
- **Backend:** Node.js with Hono or Fastify
- **Database:** PostgreSQL (with Prisma or Drizzle ORM)
- **Cache:** Redis
- **Queue:** BullMQ
- **AI:** Anthropic SDK / Vercel AI SDK
- **Infra:** Docker, Railway or Fly.io

## Data Sources
- **OpenF1 API** — real-time telemetry and timing during live sessions

## Status
Early development. Folder structure and feature scope are still being defined.
